# Rest Assured – Task 4 Scenario 2 (ReqRes API)

## Overview
This project automates REST API testing using Rest Assured and TestNG for the ReqRes API.
It demonstrates:
Data-driven testing
Conditional logic (delete if ID=3)
CLI
Assertion of response data
HTML reporting
